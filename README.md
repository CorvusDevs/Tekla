Tekla is a floating virtual keyboard for macOS that lets you type by swiping across letters or tapping keys individually. With context-aware predictions, bigram language models, and support for 14 languages — all running 100% offline.

## Features

- **Swipe-to-Type** — Slide your finger or pointer across letters to type words instantly, powered by geometric scoring and Bayesian ranking
- **Smart Predictions** — Context-aware word predictions using bigram language models and frequency data, with next-word suggestions
- **14 Languages** — Full keyboard layouts and prediction data for English, Spanish, German, French, Portuguese, Italian, and 8 more
- **100% Offline** — No internet required. All processing and language data stays entirely on your device
- **Full Desktop Layout** — Complete Mac keyboard with function keys, modifier keys, and navigation cluster in a resizable floating window
- **Lightweight** — Near-zero CPU usage when idle with an event-driven architecture

## Accessibility

Tekla is designed for users who need alternative input methods:

- **Motor Accessibility** — Full typing with any pointing device, no physical keyboard needed
- **Flexible Input** — Works with trackpads, mice, styluses, head pointers, and any pointing device
- **Fewer Keystrokes** — Smart predictions and next-word suggestions reduce the effort needed to type

## Privacy

All processing happens locally on your device. No data is ever sent anywhere. No analytics, no telemetry, no network requests.
